Changes
=======

The following is a list of changes made to the Velocity Docbook Framework (DBF) to generate Click documentation.

# Upgraded to Apache FOP v0.95, which supports soft page breaks 

  lib/fop-0.95.jar


# Upgraded to Docbook XLS project latest snapshot, which supports syntax highlighting 

  src/zip/docbook-xsl-snapshot.zip


# Includes XSLTHL syntax highlighter version 2.0.1

  lib/xslthl-2.0.1.jar
